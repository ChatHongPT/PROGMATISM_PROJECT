## 프로젝트 개요
뇌성마비 환자들의 발음 특성을 고려한 개인화된 음성 인식 프로그램을 개발하여, 기존 방식의 문제점을 해결하고 뇌성마비 환자들의 의사소통과 생활 편의성을 향상

### 프로젝트 매니저 (PM)
- **홍건하** (전북대학교)
  
### 팀원
- **배대한** (원광대학교)
- **최홍석** (전북대학교)
- **김미르** (인하대학교)
